# 智慧门禁系统

基于树莓派的智能门禁系统，支持人脸、密码、指纹、卡片等多种开锁方式，通过华为云实现远程监控与控制。

## 项目结构

```
门禁/
├── 人脸识别/          # 人脸特征训练工具
│   ├── 一体化脚本.py   # 人脸采集、训练、验证一体化工具
│   ├── dataset/        # 人脸照片数据集
│   └── 训练说明文档.md # 训练原理与使用说明
├── 上位机框架/         # 树莓派上位机界面（Tkinter）
│   ├── main.py         # 程序入口
│   ├── gui_main.py     # 主控制器
│   ├── pages/          # 页面模块
│   └── 框架说明文档.md # 框架设计说明
├── 功能汇总/           # 集成版（含MQTT通信）
├── 小程序/             # 微信小程序远程控制
│   ├── 接口实现/       # 小程序代码
│   └── 说明文档.md     # 小程序功能说明
└── 驱动/               # 硬件驱动模块
    ├── DHT11/          # 温湿度传感器
    ├── WIFI上云/       # WIFI连接与云端通信
    ├── 指令订阅/       # MQTT指令接收
    └── （ADC）D0采集/  # ADC模拟量采集
```

## 核心功能

- **人脸识别**：基于OpenCV SFace模型，支持数据增强和高精度特征匹配
- **密码开锁**：数字键盘密码验证
- **指纹开锁**：预留指纹传感器接口
- **卡片开锁**：预留NFC/RFID接口
- **远程控制**：微信小程序实时监控和控制
- **环境监测**：温度、湿度、光照、雨滴检测
- **设备控制**：继电器（门锁）、蜂鸣器控制

## 技术栈

- **硬件**：树莓派4B
- **语言**：Python 3.8+
- **界面**：Tkinter（上位机）、微信小程序（远程）
- **AI**：OpenCV DNN、SFace人脸识别
- **通信**：华为云IoT、MQTT、REST API

## 快速开始

1. **人脸训练**：运行 `人脸识别/一体化脚本.py` 采集照片并训练特征库
2. **上位机运行**：运行 `上位机框架/main.py` 启动门禁界面
3. **小程序部署**：使用微信开发者工具打开 `小程序/接口实现` 目录
